# @cyberskill/ds-cinematic

Marketing-only extension package of `@cyberskill/ds-foundations`. Used by the CyberSkill landing page (Lumi the Golden Genie).

**Lifecycle:** Experimental (will graduate to Stable 28 days bug-clean post-launch per FR-DS-009).

```ts
import { colors, motion, typography, glow, accents, LIFECYCLE_STAGE } from '@cyberskill/ds-cinematic';
```

Do NOT fork `@cyberskill/ds-foundations`. This package extends it; foundations stays the single source of truth for product UI tokens.

See: master plan §3.1, FR-DS-003, FR-DS-004..009 in `docs/feature-requests/ds/`.
